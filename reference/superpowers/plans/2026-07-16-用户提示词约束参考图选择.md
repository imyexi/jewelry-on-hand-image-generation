# 用户提示词约束参考图选择实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 `jewelry-on-hand-workflow` 在有选图提示词时执行全部条件硬匹配、无提示词时只依据适用品类与关键词相关性选择，并确保选图提示词永不进入 AIReiter Prompt。

**Architecture:** 新建独立的参考图选择约束模块，负责提示词规范化、白名单字段匹配、候选审计和稳定摘要；评分模块只负责基础门禁、提示词过滤、排序与候选不足报告。CLI 将审计文件写入 run，并把摘要绑定到候选、人工决策和生成 gate；Prompt 构建层只读取已选参考图元数据，不接收选图提示词。

**Tech Stack:** Python 3.13、标准库 `dataclasses/json/hashlib/re`、pytest、现有 `jewelry-on-hand` CLI 与 JSON run 契约。

```python
MATCH_FIELD_NAMES = (

def normalize_reference_selection_prompt(value: str | None) -> tuple[str, ...]:
    if not value:
        return ()
    normalized: list[str] = []
    for part in re.split(r"[，,；;\r\n]+", value):
        condition = part.strip()
        if condition and condition not in normalized:
            normalized.append(condition)
    return tuple(normalized)
```

```python
@dataclass(frozen=True)
class ReferenceSelectionConstraints:
    original_prompt: str
    normalized_conditions: tuple[str, ...]

    @property
    def mode(self) -> str:
        return "prompt_hard_gate" if self.normalized_conditions else "keyword_relevance_only"

@dataclass(frozen=True)
class CandidateConstraintEvaluation:
    reference_index: int
    matched: bool
    condition_matches: dict[str, tuple[str, ...]]

def reference_selection_sha256(payload: Mapping[str, Any]) -> str:
    encoded = json.dumps(
        payload, ensure_ascii=False, sort_keys=True, separators=(",", ":")
    ).encode("utf-8")
    return hashlib.sha256(encoded).hexdigest()
```

`evaluate_candidate_constraints()` 只遍历 `MATCH_FIELD_NAMES`，使用 `casefold()` 子串匹配。`build_reference_selection_audit()` 固定输出 `schema_version=1`、模式、原文、条件、三阶段候选数、逐条件命中数和逐候选证据。
