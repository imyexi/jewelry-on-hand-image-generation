# 7 月会员日项链上手图批量生成实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 7 月会员日货盘中 6 个合规项链 SKU 生成、质检、加水印并回填一张新的真人佩戴图，同时保持 `QY039` 不变并记录阻断原因。

**Architecture:** 飞书 Base 负责真实记录、附件下载和最终附件回填；本地 `jewelry-on-hand` 先完整同步线上参考图库，再通过仅在本批显式启用的 `--ignore-pending-enrichment` 排除 47 条待补全素材并写入来源快照，随后完成产品分析、参考图 Top 3、人工决策门禁、AIReiter 提交和结构化 QC；`yuanyuan-ruyi-watermark` 负责最终水印。每个 SKU 使用独立不可覆盖 run，只有 `qc.status=pass` 的结果才进入水印和飞书上传。

**Tech Stack:** PowerShell、`lark-cli base`、Python 3.11、`uv`、Pillow、`jewelry-on-hand` CLI、AIReiter `gpt_image_2`、Yuan Studio 水印脚本。
